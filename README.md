# Application-Budget-2025

## Description
Application de gestion de budget personnel pour l'année 2025. Cette application permet de suivre vos revenus, dépenses et de planifier votre budget mensuel et annuel.

## Aperçu

L'application est entièrement côté client : aucune base de données ou dépendance serveur n'est nécessaire. Les données sont sauvegardées dans le LocalStorage du navigateur.

### Ce que vous pouvez faire

- Ajouter des transactions (revenus et dépenses) avec date, catégorie, description et montant.
- Filtrer l'historique par type, mois et catégorie.
- Visualiser les totaux de revenus, de dépenses et votre solde actuel.
- Définir un budget annuel ainsi que des budgets mensuels et suivre la progression.
- Afficher des graphiques (Chart.js) pour l'évolution mensuelle et la répartition des dépenses par catégorie.

## Fonctionnalités prévues
- 📊 Suivi des revenus et dépenses
- 📈 Visualisation des données avec graphiques
- 💰 Catégorisation des transactions
- 📅 Planification budgétaire mensuelle/annuelle
- 📱 Interface responsive (web et mobile)
- 💾 Sauvegarde locale des données

## Technologies
- Frontend: HTML, CSS, JavaScript
- Graphiques: Chart.js
- Stockage: LocalStorage

## Installation et utilisation
1. Cloner le repository
2. Ouvrir `index.html` dans votre navigateur
3. Commencer à saisir vos données financières
